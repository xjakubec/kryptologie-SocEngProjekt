Krok 1

GitHub Pages – hosting zdarma

Nejjednodušší způsob jak dostat index.html online za 0 Kč.

1
Vytvoř GitHub účet a repository
Zdarma▶
  • Jdi na github.com → Sign up (stačí e-mail)
  • Po přihlášení klikni + New repository
  • Název: soceng-projekt, nastav na Public
  • Zaškrtni Add a README file → Create repository
2
Upload souborů
Snadné▶
  • V repo klikni Add file → Upload files
  • Přetáhni oba soubory: landing_page.html + skoleni.html
  • Důležité: landing_page.html přejmenuj na index.html před uploadem
  • Klikni Commit changes
!skoleni.html nech pojmenovaný jako skoleni.html – bude dostupný na /skoleni.html a sdílíš ho až po debriefingu.
3
Zapnout GitHub Pages
60 sekund▶
  • V repository → Settings → Pages (levé menu)
  • Branch: main, složka: / (root) → Save
  • Počkej ~60s → zobrazí se zelená zpráva s URL
Výsledné URL pro každou skupinu:
# Skupina 1 – Prarodiče (SMS) https://tvoje-jmeno.github.io/soceng-projekt/?skupina=1 # Skupina 2 – Rodiče (e-mail) https://tvoje-jmeno.github.io/soceng-projekt/?skupina=2 # Skupina 3 – Vrstevníci (Messenger/IG) https://tvoje-jmeno.github.io/soceng-projekt/?skupina=3 # Skolení (po debriefingu) https://tvoje-jmeno.github.io/soceng-projekt/skoleni.html
✓Tyto URL jsou dlouhé – pro SMS je zkrať přes bitly.com nebo tinyurl.com (zdarma). Pro e-mail a chat délka nevadí.
Krok 2

Měření prokliků – Cloudflare Analytics

Bezplatná analytika bez cookies. Vidíš přesný čas každého kliknutí, zařízení a skupinu.

1
Registrace Cloudflare Web Analytics
Zdarma, bez karty▶
  • Jdi na dash.cloudflare.com → Sign up
  • V levém menu: Web Analytics → Add a site
  • Zadej URL tvého GitHub Pages (tvoje-jmeno.github.io/soceng-projekt)
  • Cloudflare vygeneruje krátký JS snippet s tokenem
2
Přidat script do index.html
2 minuty▶

Na GitHubu otevři index.html → klikni na tužku (edit). Před </body> vlož:

<script defer src='https://static.cloudflareinsights.com/beacon.min.js' data-cf-beacon='{"token": "TVŮJ_TOKEN_ZDE"}'></script>
ℹCloudflare Web Analytics nesbírá IP adresy ani cookies, je plně GDPR compliant. Vidíš: čas návštěvy, zemi, zařízení (mobile/desktop), která URL parametr byl použit.
3

Útok č. 1 – Prarodiče

SMS útok – Fio Banka

Bez spoofingu – a proč to pro projekt nevadí.

1
Proč vlastní číslo funguje stejně dobře
▶
  • Prarodiče typicky nečtou číslo odesílatele – reagují na obsah. Toto je samo o sobě silný závěr projektu.
  • Pokud číslo poznají → záměrně ho neznají (babička ho nezná nazpaměť) → experiment stále funguje.
  • Pokud zareagují i přes to, že číslo neznají → dokazuješ zranitelnost vůči neznámým číslům.

Záložní možnost bez spoofingu: Předplacená SIM karta (Vodafone/T-Mobile ~99 Kč v trafice). Odešleš z čísla které babička nezná – přidá autentičnost „číslo banky". Plně legální, žádný spoofing.

2
Finální útočný text SMS
▶
Fio Banka: Na vasem uctu bylo zaznamenano prihlaseni z noveho zarizeni (Praha, Chrome). Pokud jste to nebyli Vy, okamzite potvrdte identitu zde: https://bit.ly/ZKRACENA-URL Pokud jste prihlaseni provedli Vy, tuto zpravu ignorujte.
!Klíčový detail: „Praha, Chrome" – konkrétní lokace + prohlížeč vytváří iluzi legitimního bezpečnostního systému. Poslední věta snižuje podezření – skutečné bankovní alerty ji také obsahují.
✓URL zkraťte přes bitly.com – zkrácený odkaz vypadá profesionálněji v SMS a neodhaluje github.io. Bitly link nastav na přesměrování na index.html?skupina=1.
Útok č. 2 – Rodiče

Mailjet – podvodný e-mail od "Finanční správy"

Mailjet umožňuje nastavit libovolné "Od:" jméno i adresu. E-mail vypadá jako od státní instituce.

1
Proč Mailjet, ne Gmail
▶
Gmail – nevhodné
Příjemce vidí celou adresu „Jméno <skutecna@gmail.com>". Gmail přidává varování „via gmail.com". Žádná kontrola nad hlavičkami.
Mailjet – ideální
Nastavíš přesné „Od:" jméno. Free tier = 200 e-mailů/den. Profesionální doručení, bez spam varování.
2
Registrace a nastavení Mailjet
Free tier stačí▶
  • Jdi na mailjet.com → Sign Up Free
  • Ověř svůj skutečný e-mail (pro přihlášení do Mailjet)
  • Dashboard → Account → Sender domains & addresses
  • Klikni Add a Sender Address (ne doménu, jen adresu)
  • Zadej podatelna.fin.sprava@gmail.com (vytvoř tuto Gmail adresu předem)
  • Mailjet pošle ověřovací e-mail na tuto adresu – ověř ji kliknutím
ℹNepotřebuješ vlastní doménu. Gmail adresa podatelna.fin.sprava@gmail.com jako sender adresa v Mailjet stačí. Příjemce uvidí jako „Od:" pouze to co nastavíš v kampani.

Útok č. 3 – Vrstevníci

Messenger / Instagram – obejití filtru žádostí

Jak dostat zprávu přímo do hlavního chatu bez čekání na přijetí žádosti.

1
Jak filtr žádostí funguje a jak ho obejít
▶

Facebook/Instagram přesouvá zprávy od neznámých do „Žádostí o zprávy" bez notifikace. Čtyři způsoby jak to obejít:

A – IG Story reakce ⭐
Zareaguj na jejich IG story emotikonem → DM jde přímo do hlavního chatu. Nejspolehlivější metoda.
B – Skupinový chat
Přidej cíl do existující skupiny kde jste oba → zprávy tam jdou přímo, bez filtru.
C – Messenger.com na PC
Méně striktní filtrování než mobilní aplikace při sdílené skupině nebo kontaktu.
D – Dočasné přátelství
Falešný profil → žádost o přátelství → cíl přijme → odešli zprávu → zruš přátelství.
2
Doporučený postup: IG Story reakce
Nejspolehlivější▶
  • Vytvoř nový IG účet se jménem spolužáka (nebo kamaráda)
  • Přidej profilovku + 2-3 příspěvky aby profil vypadal reálně
  • Sleduj profil cíle – čekej na story (nebo sleduj jejich aktivitu)
  • Zareaguj na story emotikonem → automaticky se otevře DM v hlavním chatu
  • V té DM konverzaci okamžitě odešli baiting zprávu s odkazem
↑Pokud cíl nemá IG nebo stories nepřidává: použij metodu B (skupinový chat) nebo D (dočasné přátelství přes falešný FB profil).
3
